# PM Agent 今後のロードマップ案

プロジェクトを現在の「モニタリングのみのMVP」から、より実用的で強力な製品へ進化させるための主要なステップです。

## 1. インタラクティブ機能の強化（HubSpotへの書き戻し）
現在は閲覧とAI解析が中心ですが、PM Agentから直接アクションを実行できるようにします。
- **アラートの完全同期**: アラートの「却下」や「解決」をHubSpotのメモやカスタムプロパティに反映。
- **AIメール下書き生成**: 低スコアの案件に対し、Geminiが状況を考慮した返信メールのドラフトを作成。
- **ステータス更新**: PM Agent上でプロジェクト（取引）のステージを変更し、HubSpotへ自動反映。

## 2. AI 解析エンジンの高度化
より深い洞察と精度の高いリスク予測を実現します。
- **長期的な感情トレンド分析**: 1通のメールだけでなく、過去数週間の感情スコアの推移をグラフ化し、悪化の兆候を検知。
- **タスク自動抽出**: メール内容から「次に私たちがすべきこと（Next Action）」をAIが自動的に抽出し、タスクリストに追加。
- **複数チャネルの統合**: HubSpot上のノート、メールに加え、Slackなどの他ツールからの情報も解析対象に含める。

## 3. 通知およびコラボレーション機能
「見逃さない」をさらに自動化します。
- **Slack/Webブラウザ通知**: 重大なリスク（At Risk判定）が検出された際、即座に担当者へ通知を飛ばす。
- **チームビューの最適化**: 複数のPMがそれぞれの担当案件のみを効率的に管理できるフィルタリングとパーソナライズ。

## 4. プロダクション環境への準備
実際の業務で安定して運用するための基盤強化です。
- **認証・認可 (Auth)**: ユーザーログイン機能の実装と、ユーザーごとのHubSpot権限の管理。
- **パフォーマンス最適化**: HubSpot APIのレート制限を考慮したデータキャッシング機構。
- **テストの自動化**: AI解析ロジックやAPI連携部分のユニットテスト・結合テストの拡充。
